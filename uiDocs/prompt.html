    # Attributes

    参数 说明 类型 可选值 默认值
    title 内容 string - —
    type 类型 string success/warn/info/error warn
    icon 图标类名 string — —
    closable 是否可关闭 boolean — true
    center 是否居中 boolean — false
    openkey 为是否重新打开，提供key值缓存name string — -
