    # Icon 图标

    提供了一套常用的图标集合。

    # 使用方法

    直接通过设置类名为 icon-iconName 来使用即可。例如：

    <i class="j-icon icon-reeor"></i>
<i class="j-icon icon-share"></i>
<i class="j-icon icon-set"></i>
<j-button type="primary" icon="icon-editor1">修改</j-button>
